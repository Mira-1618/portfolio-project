My Board Game Library (WIP - Coming Soon!)
An app to track what's on your shelf and when you've last played!

So far, My Board Game Library consists of a handful of sketches and a few python files, but I have big plans for it. By the time I am finished, MBGL should be able to track the games users input, pull information via "Board Game Geek" API's, and randomly select a game for users to play next based upon what is currently in their library and a few provided filters.

  • Consumes External APIs
  • Mobile App Planned
